## 📌 Visão Geral

> Aplicação web desenvolvida para simular o funcionamento de uma Unidade de Pronto Atendimento (UPA), permitindo o cadastro de pacientes, realização da classificação de risco, organização automática da fila de atendimento e chamada dos pacientes conforme o nível de prioridade. O projeto tem como objetivo aplicar conceitos de desenvolvimento Full Stack, arquitetura em camadas, APIs REST e banco de dados relacionais durante a formação do Programa Jovem Programador (SEPROSC/SENAC-SC).

---

## 🎯 Objetivo

> Desenvolver uma aplicação web capaz de gerenciar o fluxo de atendimento de uma Unidade de Pronto Atendimento, tornando mais eficiente o processo de triagem, classificação de risco, organização da fila e acompanhamento dos atendimentos.

## ⚙️ Funcionalidades

### 🩺 Gestão de Pacientes

* Cadastro de pacientes
* Atualização de dados
* Consulta de pacientes
* Exclusão de registros

### 🚨 Classificação de Risco

* Registro da triagem
* Definição do nível de prioridade
* Organização automática da fila conforme a classificação

### 📢 Atendimento

* Chamada de pacientes
* Controle da fila de atendimento
* Histórico de atendimentos
* Alteração do status do paciente
