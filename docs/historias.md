# Histórias de Usuário

## HU01 — Login
Como funcionário, quero entrar no sistema para acessar minha área.

## HU02 — Funcionários
Como administrador, quero cadastrar funcionários e atribuir cargos.

## HU03 — Atendimento
Como recepcionista, quero cadastrar um paciente e gerar uma senha.

## HU04 — Triagem
Como enfermeiro, quero registrar sinais e classificação de risco.

## HU05 — Consulta
Como médico, quero registrar o prontuário e finalizar o atendimento.
